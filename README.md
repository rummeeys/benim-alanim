1. FILES ON THE DISK
--------------------------------------------------------------------------------
The directory structure of the "Benim Alanım" software package consists of the 
following core source, asset, and configuration files:

- /src/app/             : Next.js application pages, routing links, and layouts.
- /src/components/      : Modular UI components (games, pop-up modals, dashboard).
- /src/middleware.js    : Custom route guards managing secure student sessions.
- /public/assets/       : SVG vector graphics and character design assets.
- /public/sounds/       : Localized scenario voiceover files (.m4a format).
- /public/videos/       : Educational animated video sequences (.mp4 format).
- package.json          : Project dependency listings and build script codes.
- tailwind.config.js    : Custom interface design styling parameters.

--------------------------------------------------------------------------------
2. HARDWARE REQUIREMENTS
--------------------------------------------------------------------------------
To run, render, and interact with the software layers smoothly, the system configuration 
must satisfy the following minimum requirements:

- CPU                   : Intel Core i3 / Apple Silicon M1 or higher
- RAM Capacity          : Minimum 4 GB RAM (8 GB recommended)
- Disk Capacity         : 500 MB minimum available local disk storage
- Input Devices         : Standard Mouse, Trackpad, or Interactive Smartboard
- Display / Graphics    : Graphics card supporting 1280x720 minimum screen resolution

--------------------------------------------------------------------------------
3. SOFTWARE REQUIREMENTS
--------------------------------------------------------------------------------
The software workspace, compilers, and libraries necessary to run and link the 
application include:

- Operating System      : Microsoft Windows 10/11, macOS 12+, or Linux
- Execution Runtime     : Node.js Runtime Environment (v18.0.0 or higher)
- Package Manager       : Node Package Manager (npm) / Yarn Toolchain
- Core Platform Engine  : Next.js Production Web Framework (v14.x)
- CSS Utility Layer     : Tailwind CSS Processing Framework
- External API Client   : Supabase JS Client Client Engine
